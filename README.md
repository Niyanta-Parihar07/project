# Number Guessing Game 🎮

A fun command-line game built in Python where the computer secretly picks a 
random number and you have to guess it within 5 attempts!

## About the project
This was my first Python project, built to practice core programming concepts
like functions, loops, conditionals, and the random module.

## How to play
1. The computer randomly picks a number between 1 and 100
2. You get a maximum of 5 attempts to guess it
3. After each wrong guess, you get a hint — too high or too low
4. Guess correctly within 5 attempts to win!

## How to run
Make sure Python is installed, then run:
```
python guessing_game.py
```

## Features
- Random number generation using Python's random module
- Maximum attempt limit (5 tries)
- Hints after every wrong guess
- Game over message reveals the correct answer if you lose

## What I learned
- Defining and calling functions in Python
- while loops and break statements
- if / elif / else conditions
- Taking user input and comparing values
- Using the random module
